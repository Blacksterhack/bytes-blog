IPCAM: Ver camaras abiertas a traves de la IP.






IPCAM es un script codificado en python, el cual permite la obtencion de direcciones IP, de camaras que han sido vulneradas o estan abiertas. para el total funcionamiento de este script es necesario que la persona este conectada a internet, pues de esta manera obtendra las direcciones IP, con las cuales podra acceder a distintas camaras en el mundo.

Instalacion en termux:

git clone https://github.com/Blacksterhack/IPCAM
